# 🛍️ Customer Shopping Behavior Analysis
# 🛍️ Análisis del Comportamiento de Compra de Clientes

---

## 🇬🇧 English

### 📌 Overview
This project analyzes customer shopping behavior using Python, PostgreSQL, SQL, and Power BI to identify purchasing patterns and generate business insights. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard development, and reporting.

### 📂 Dataset
The dataset contains customer shopping information such as:

- Customer demographics
- Product categories
- Purchase amount
- Review ratings
- Payment methods
- Discounts and promo codes
- Shipping types
- Purchase frequency

### 🛠️ Tools & Technologies

- Python (Pandas, NumPy)
- PostgreSQL
- Power BI
- Gamma
- Git & GitHub

### 📈 Project Workflow

1. Loaded the dataset using Python.
2. Performed Exploratory Data Analysis (EDA).
3. Cleaned and transformed the data.
4. Imported the cleaned dataset into PostgreSQL.
5. Answered business questions using SQL.
6. Built an interactive Power BI dashboard.
7. Designed the final presentation using Gamma.

### 📊 Dashboard

The Power BI dashboard includes:

- Sales Overview
- Customer Demographics
- Product Performance
- Customer Segmentation
- Purchase Trends
- Discount Analysis

### 📌 Key Results

- Identified the best-selling products.
- Analyzed customer purchasing behavior.
- Measured the impact of discounts.
- Compared product categories.
- Generated business insights through data visualization.

### 🚀 How to Run

1. Clone this repository.
2. Install the required libraries.
3. Run the Python scripts.
4. Import the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI dashboard.

## 🇪🇸 Español

### 📌 Descripción General

Este proyecto analiza el comportamiento de compra de los clientes utilizando Python, PostgreSQL, SQL y Power BI para identificar patrones de compra y generar información útil para la toma de decisiones. El flujo de trabajo incluye carga de datos, análisis exploratorio (EDA), limpieza de datos, consultas SQL, creación de dashboards y elaboración de un reporte.

### 📂 Conjunto de Datos

El conjunto de datos contiene información sobre:

- Datos demográficos de clientes
- Categorías de productos
- Monto de compra
- Calificaciones de productos
- Métodos de pago
- Descuentos y códigos promocionales
- Tipos de envío
- Frecuencia de compra

### 🛠️ Herramientas y Tecnologías

- Python (Pandas, NumPy)
- PostgreSQL
- Power BI
- Gamma
- Git y GitHub

### 📈 Flujo del Proyecto

1. Carga del conjunto de datos con Python.
2. Análisis Exploratorio de Datos (EDA).
3. Limpieza y transformación de datos.
4. Importación del conjunto de datos limpio a PostgreSQL.
5. Desarrollo de consultas SQL para responder preguntas de negocio.
6. Creación de un dashboard interactivo en Power BI.
7. Diseño de una presentación final en Gamma.

### 📊 Dashboard

El dashboard incluye:

- Resumen de ventas

### 📌 Resultados Principales

- Identificación de los productos más vendidos.
- Análisis del comportamiento de compra de los clientes.
